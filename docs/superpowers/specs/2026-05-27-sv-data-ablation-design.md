# SenseVoice 微调数据量实验设计

## 1. 目标

验证 SV 模型在不同数据量及数据增强策略下微调后的 CER 变化。

## 2. 实验组设计

| 实验组 | 数据量 | 说明 |
|--------|--------|------|
| 1 | 25% | 随机子采样 |
| 2 | 50% | 随机子采样 |
| 3 | 75% | 随机子采样 |
| 4 | 100% + 合成噪音 (A) | 原始样本 + 白噪音叠加 → 200% |
| 5 | 100% + 真实噪音 (B) | 原始样本 + DEMAND 噪音 → 200% |

> 基准组（100% 原始数据）已运行，不重复。

## 3. 数据增强方案

### 3.1 合成噪音 (A)
- 使用 `torch.randn` 生成随机白噪音
- 按信噪比（SNR）叠加到原始音频
- SNR 范围：10dB ~ 20dB（随机选择）

### 3.2 真实噪音 (B)
- 使用 DEMAND 噪音数据集
- 按相同 SNR 范围叠加

## 4. 训练配置（保持不变）

| 参数 | 值 |
|------|-----|
| 模型 | SenseVoice-Small (LoRA) |
| LoRA rank | 8 |
| LoRA alpha | 16 |
| 学习率 | 2e-4 |
| max_epoch | 10 |
| batch_size | 6000 |
| 验证集 | val.jsonl（不变） |

## 5. 评估指标

- **CER（字错误率）**：在验证集上计算
- 对比各实验组与基准组的 CER 差异

## 6. Notebook 结构

```
06_data_ablation.ipynb
├── 1. 环境检查
├── 2. 数据准备
│   ├── 2.1 生成子采样数据（25%/50%/75%）
│   ├── 2.2 生成合成噪音增强数据（A）
│   └── 2.3 生成真实噪音增强数据（B）
├── 3. 训练（5 个实验组逐一训练）
├── 4. CER 评估
│   └── 汇总对比表格 + 可视化
└── 5. 结论分析
```

## 7. 输出

- 5 个模型的 checkpoint 分别保存到独立目录
- 最终生成 CER 对比表格和柱状图
